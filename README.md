# Job Search Documentation (Japan)

This repository describes the process of transitioning to a career as a technical writer in Japan for a foreign candidate.
Rather than offering motivational stories, the series analyzes the job search as a system:

- Assumptions.
- Constraints.
- Iteration cycles.
- Portfolio refinement.
- Observing the hiring process.

This article series can be useful for any foreign professional who wants to build a career in Japan, or is at least considering it. If you have any criticisms, comments, or suggestions, please contact me on LinkedIn: [Igor Rosliakov](https://www.linkedin.com/in/igor-rosliakov-techwriter/)
