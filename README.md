# Job Search Documentation (Japan) 🇯🇵

## What does this documentation include

This repository describes the process of transitioning to a career as a technical writer in Japan for a foreign candidate.
Moreover, the documentation can be used by any specialists, students, and other foreign citizens who want to find their place in the Japanese labor market.

Rather than offering motivational stories, the series analyzes the job search as a system:

- Assumptions.
- Constraints.
- Iteration cycles.
- Portfolio refinement.
- Observing the hiring process.

## How it was written and how to perceive it

While the material in these articles is intended to help expats ease the discomfort and pain of immersion in the Japanese labor market, it is not a strict set of rules—it is simply a collection of notes I have made while living and job hunting in Japan.

I ask that you take these articles with a grain of salt, as your experience may differ from mine. In fact, taking these documents as strict recommendations can be harmful and even dangerous.
